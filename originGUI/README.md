# Cheat GUI

This is the main cheat menu GUI for Prodigy Origin.

<br><br>

## Usage
Install the hacks, the dropdown should appear in the top left corner after a moment.

<br><br>


## Development
Install dependencies:
```shell
pnpm run setup
```
<br>

Compile the menu:
```shell
pnpm run build
```
<br><br>

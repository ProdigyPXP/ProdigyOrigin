# Manual Install — Opera

1. Download the latest release from the [Releases page](https://github.com/ProdigyPXP/ProdigyMathGameHacking/releases/latest).
2. Extract the ZIP file.
3. Open Opera and go to `opera://extensions/`.
4. Enable **Developer Mode**.
5. Click **Load unpacked** and select the extracted folder.
6. Navigate to [math.prodigygame.com](https://math.prodigygame.com/) to verify the extension is working.

> **Note:** Opera supports Chrome extensions. You may also install directly from the Chrome Web Store.

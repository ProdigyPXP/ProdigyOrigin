# Prodigy Origin Privacy Policy

_Last updated: April 7, 2026_

Prodigy Origin is a free, open source browser extension.

## Data Collection

Prodigy Origin does **not** collect, store, or transmit any personal data.

The extension operates entirely locally in your browser. No analytics, no telemetry, no user tracking.

## Permissions

- **declarativeNetRequest** — blocks/redirects the original game file and custom images
- **storage** — stores your custom game/mod bundle URL overrides locally in `chrome.storage.local`
- **Host permissions** (`*.prodigygame.com`, `raw.githubusercontent.com`) — required to intercept game requests and fetch patched assets from GitHub

## Third-Party Services

The extension fetches patched game files from `raw.githubusercontent.com` (GitHub). GitHub's own privacy policy applies to those requests.

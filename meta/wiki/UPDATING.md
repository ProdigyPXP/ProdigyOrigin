# Updating Prodigy Origin

## Web Store Installs

If you installed via a web store, the extension updates automatically.

## Manual Installs

1. Download the latest release from [GitHub Releases](https://github.com/ProdigyPXP/ProdigyMathGameHacking/releases/latest).
2. Extract and replace your existing extension folder.
3. Go to your browser's extensions page and click **Reload** on Prodigy Origin.
